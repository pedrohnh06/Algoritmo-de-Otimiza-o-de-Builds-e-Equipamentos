# RPG Build Simulator

Projeto desenvolvido em Python para praticar Programação Orientada a Objetos.

## Funcionalidades

- Criação de personagens
- Sistema de equipamentos
- Chance de acerto crítico
- Simulação de ataques
- Comparação automática de armas
- Cálculo de dano médio

## Tecnologias

- Python 3

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Classes e objetos
- Métodos
- Listas
- Simulação probabilística
- Estruturas de repetição

## Como executar

python main.py
